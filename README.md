# Kropp-Fitness 🏋️‍♀️

A clean, responsive one-page website for a fitness club, built based on a Figma design.

## 📌 About the Project

This project is a static landing page created as a layout practice from an existing Figma design.

- Fully implemented using **HTML** and **CSS**
- No JavaScript used
- Focus on layout accuracy, responsiveness, and smooth CSS animations
- Designed as a modern fitness club website

## 🎯 Features

- Responsive layout (desktop / tablet / mobile)
- CSS-based animations and transitions
- Semantic HTML structure
- Clean and minimal code

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, animations)

## 🚀 How to Run

Since this is a static project, no installation is required.

1. Download or clone the repository
2. Open `index.html` in your browser

## ✏️ Notes

This project was created for practice purposes, focusing on:

- Translating design (Figma) into code
- Improving layout and styling skills
- Working with pure CSS animations without JavaScript
